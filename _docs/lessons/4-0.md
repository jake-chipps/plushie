## Objectives
- Finish the plushie.
- Test the computational artifact and debug.

## Materials
The following materials are required for this lesson:
- Constructed project so far
- Regular or embroidery thread
- Sewing needle
- Fluffy stuffing
- Computer with internet
- USB to microUSB

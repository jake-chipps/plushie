This curriculum is an introduction to programming physical computational artifacts.

### 2 Program the Lights
- Program the LEDs.
- Debug the code.

### 3 Program the Music
- Program the music.
- Debug the code.

### 4 Make the Plushie
- Construct the final project.
- Test.

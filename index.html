DIY Light-Up Musical Plushie



An E-Textiles Project

DIY Electronic Plushie

This DIY guide is a step-by-step process for creating plushie with LEDs that display a light pattern and plays a musical melody. No programming experience or sewing experience required before starting this project. Anyone age 8 and older can create this project.

Integratable

This project can be integrated into any subject matter in formal or informal education settings.

Project Based

This is a single project that enables the user to construct of a computational artifact based on their designs. Bringing together makerspaces and constructionism, this DIY project grounds learning of computer science and engineering in active, hands-on learning.

For Everyone

This curiculum is a step-by-step process in creating circuits using a microcontroller and developing computational artifacts using input and output devices. This is a great place to start if you have never worked with computing systems or circuitry before. But even those familiar with circuits and programming circuits may still find value in this project. The goal is to be able to create your own projects after completing this project.
